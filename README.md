<h1>YouTube Playlist Duration &amp; Timetable Calculator</h1>

<p>A robust browser extension that streamlines the planning of video-based learning by calculating total playlist durations and generating customizable viewing schedules.</p>

<h2>Core Functionality</h2> <ul> <li><strong>Automated Duration Calculation:</strong> Parses the YouTube interface to aggregate the total time of all videos in a playlist, displaying the results in Days, Hours, Minutes, and Seconds.</li> <li><strong>Dynamic Timetable Generation:</strong> Allows users to input a daily time commitment in minutes to receive a comprehensive, day-by-day distribution of videos.</li> <li><strong>Video Continuity Logic:</strong> Features an algorithm that accounts for videos exceeding the daily time limit, instructing the user where to pause and resume the following day.</li> <li><strong>Data Portability:</strong> Includes a dedicated export function that formats the generated timetable for printing or digital archiving.</li> <li><strong>Contextual Activation:</strong> The extension intelligently detects the current URL to ensure the timetable tools are only active when a valid YouTube playlist is detected.</li> </ul>

<h2>Technical Architecture</h2> <ul> <li><strong>Platform:</strong> Built on the Chromium Manifest V3 standard for enhanced security and optimized resource management.</li> <li><strong>Data Acquisition:</strong> Utilizes non-invasive DOM scraping via content scripts to identify video titles and time-status elements without requiring external API quotas.</li> <li><strong>Asynchronous Communication:</strong> Implements a message-passing architecture between the popup interface, background service workers, and active tab content scripts.</li> <li><strong>Design:</strong> Styled with a professional CSS framework featuring responsive day-cards and high-contrast typography for readability.</li> </ul>

<h2>Installation Instructions</h2> <ol> <li>Clone or download the repository source files.</li> <li>Navigate to <code>chrome://extensions/</code> in your browser.</li> <li>Enable <strong>Developer mode</strong> via the toggle in the upper-right corner.</li> <li>Select <strong>Load unpacked</strong> and choose the directory containing the extension files.</li> </ol>

<h2>Usage Guide</h2> <ol> <li>Open any public YouTube playlist page.</li> <li>Launch the extension from the browser toolbar.</li> <li>Review the <strong>Total Duration</strong> displayed in the primary dashboard.</li> <li>To create a schedule, enter your available daily minutes in the <strong>Generate Timetable</strong> section and click <strong>Generate</strong>.</li> <li>Use the <strong>Export Timetable</strong> button to save a copy of your schedule.</li> </ol>
